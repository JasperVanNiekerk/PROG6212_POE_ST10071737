PROG6212_POE_ST10071737
This is a private repository for the project of PROG6212 course at Varcity Collage. The project is a desktop application built on .NET framework using C# language. The application is a Student Time Manager.

Installation
To install the application, you need to have .NET framework installed on your computer. You can download it from here. Then, you can clone this repository or download the zip file and extract it. You will find an executable file named PROG6212_POE_ST10071737.exe in the bin folder. Double-click on it to run the application.

Usage
The Application is fairly strait forward to use please read any Instructions displayed on screen and tooltips for the best user experince please turn up your devise's folume
